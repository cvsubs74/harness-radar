# Decisions

Architecture Decision Records (ADRs). One file per significant decision.

Filename: `NNNN-<short-topic>.md`, zero-padded sequential.

Use the format from `.claude/agents/architect.md`:

```
# NNNN — topic

**Status:** accepted | superseded by NNNN | deprecated
**Date:** YYYY-MM-DD

## Context
## Decision
## Alternatives considered
## Consequences
```

Reserve `0001-stack.md` for the initial stack choice (written during `/kickoff`).
